🌀 KDE Plasma Cheat Sheet 🌀

🚀 Starting and Stopping Applications

Meta → Open the Application Launcher

Alt + F4 → Close

Ctrl + Q → Quit

Ctrl + Alt + t → Open Terminal

🪟 Moving Around

Ctrl + F10 → Present Windows

Ctrl + F12 → Show Desktop / Restore

Alt + Tab + ←/→ → Walk through Windows

Alt + F3 → Window Operations Menu

Meta + Alt + ↑/↓/←/→ → Switch Window

🔍 Panning and Zooming

F11 → Full Screen

Meta + ←/→ → Move Window

Meta + PgUp → Maximize / Restore

Meta + PgDown → Minimize

Meta + Left Mouse Button → Move Window

Meta + Right Mouse Button → Resize Window
